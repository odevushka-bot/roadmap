# Задача: собрать и задеплоить приложение "Дорожная карта — 21 день"

## Что это
Готовое React-приложение (Vite + Tailwind v4 + lucide-react). Интерактивный
чек-лист курса на 21 день. Вход по email (без пароля, просто идентификатор),
прогресс сохраняется в localStorage под этим email.

Код уже написан и рабочий — я (Claude в чате) уже собирала его и проверяла
в браузере. Ничего придумывать заново не нужно, только:

## Что нужно сделать
1. `npm install`
2. `npm run build` — проверить, что сборка проходит без ошибок
3. Инициализировать git-репозиторий (если ещё не инициализирован), закоммитить
4. Создать репозиторий на GitHub и запушить туда (`gh repo create` или через API)
5. Задеплоить на Netlify:
   - Через Netlify CLI (`netlify deploy --prod`), ИЛИ
   - Через Netlify MCP-коннектор, если он подключён в этой сессии, ИЛИ
   - Через подключение GitHub-репозитория к Netlify (Continuous Deployment)
6. В конце — вывести пользователю финальную живую ссылку (`*.netlify.app`
   или кастомный домен, если он есть)

## Важные детали, которые нельзя менять без запроса
- Цветовая палитра — все цвета заданы через inline-style в `src/App.jsx`
  (объект `C` в начале файла), НЕ через Tailwind arbitrary classes
  (`bg-[#...]`) — они не гарантированно компилируются во всех окружениях,
  поэтому специально сделано через inline style. Не переписывать обратно.
- `storage` — самодельный адаптер в начале `App.jsx`: использует
  `window.storage` (для среды артефактов Клода), иначе — `localStorage`
  (для обычного сайта). Это то, что реально хранит прогресс пользователя.
  Не убирать fallback на localStorage — без него на реальном сайте всё
  сломается.
- Обложки уроков 1–3 — закодированы как base64 (`COVER_1`, `COVER_2`,
  `COVER_3` в начале файла) — это специально, чтобы не зависеть от внешнего
  хостинга картинок (внешняя ссылка на Tilda CDN не грузилась внутри
  артефакта Клода). Можно оставить как есть или, если нужно снизить размер
  бандла, вынести в `/public` — но тогда нужно поправить пути.
- Дни 4–21 (кроме финального) сейчас содержат ЗАГЛУШКИ (иконка + placeholder-
  задания "задание 1, 2, 3, 4"). Это осознанно — реальный контент ещё не
  прислан заказчицей. Явно пометь в чате пользователю, что это заглушки,
  если он не в курсе.
